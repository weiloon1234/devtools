# Devs Tools

A versatile operational suite for developers, featuring AI asset generation, data visualization, and utility tools.

## Features
- **AI Identity Generator**: Create avatars and logos with Gemini 3 Pro.
- **QR Tools**: Generate and scan QR codes.
- **Image Tools**: Resize, crop, and remove backgrounds.
- **JSON Translator**: Translate JSON content while preserving structure.
- **Data Viewer**: Visualize complex data structures.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deployment
Deploy to Cloudflare Pages:
`npm run deploy`
